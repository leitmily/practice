# 抓取淘宝美食信息

Selenium+Chrome抓取淘宝美食

* 使用Selenium+Chrome访问网页

* 使用'--headless'参数可实现无可视化窗口访问

* 使用浏览器中的CSS选择器选取信息

* 使用pyquery模块匹配带爬取的信息

* 使用mongodb存储解析的结果
