# 今日头条图集爬取

分析Ajax抓取今日头条街拍美图

* 使用requests模块请求数据并获取网页源码

* 使用json模块解析json数据

* 使用BeautifulSoup模块解析目标页面

* 使用正则表达式匹配图集地址

* 使用mongodb存储解析的结果

* 下载的图集保存在本地

* 使用进程池模块实现多进程爬取
